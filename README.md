# C-Academy: Princípios Técnicos em Cibersegurança
## Laboratório 2
### Introdução

**Este segundo laboratório tem os seguintes objetivos:**
- Analisar programas que usam sockets para comunicação cliente/servidor;
- Realizar operações básicas na plataforma GitHub para partilha de código;
- Aplicar proteções criptográficas para proteger e desproteger ficheiros, via ferramenta OpenSSL e, opcionalmente, através de bilioteca criptográfica em Python.
